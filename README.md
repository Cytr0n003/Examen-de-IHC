# Proyecto EcoBarrio

Plataforma web para la gestión comunitaria de residuos y reportes de puntos críticos en el vecindario.

## Autor
**Nombre:** Benjamin
**Curso:** Desarrollo Web
**Fecha:** Diciembre 2025

## Descripción
EcoBarrio permite a los vecinos:
- Reportar acumulación de basura.
- Ver mapas de zonas críticas.
- Contactar a brigadistas ambientales.
- Aprender sobre reciclaje.

## Tecnologías Usadas
- HTML5 (Estructura semántica)
- CSS3 (Diseño responsivo y variables)
- JavaScript (Validación de login, buscador y animaciones)

## Instrucciones de Uso
1. Abrir el archivo `login.html`.
2. Ingresar cualquier correo.
3. **Contraseña de acceso:** `ok`
4. Navegar por las secciones del menú.
